# Scraping for news on the web.  

This module is a tool for the extraction of journalistic news on the web, implemented for:  
- El Colombiano: https://www.elcolombiano.com/      
- El Tiempo: https://www.eltiempo.com/  
- El Espectador: https://www.elespectador.com/noticias  
- Xataka: https://www.xataka.com/  
- Dinero: https://www.dinero.com/  

# Dependencies
- Python 3.6.5  
- Requests 2.18.4  
- BS4 4.6.0   
- datetime
